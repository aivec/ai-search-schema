# Sample JSON-LD Output / サンプル JSON-LD 出力

This document shows example JSON-LD outputs generated by the AVC AEO Schema plugin for different scenarios.

このドキュメントでは、AVC AEO Schema プラグインが生成する JSON-LD の出力例を紹介します。

---

## 1. Organization + WebSite (Company/Brand Site)

When `entity_type` is set to "Organization", the plugin outputs a unified @graph containing Organization, WebSite, and WebPage schemas.

`entity_type` が「Organization」の場合、Organization、WebSite、WebPage スキーマを含む統合 @graph を出力します。

```json
{
  "@context": "https://schema.org",
  "@graph": [
    {
      "@type": "Organization",
      "@id": "https://example.com/#organization",
      "name": "Acme Corporation",
      "url": "https://example.com",
      "logo": {
        "@id": "https://example.com/#logo"
      },
      "sameAs": [
        "https://twitter.com/acme",
        "https://www.facebook.com/acme",
        "https://www.linkedin.com/company/acme"
      ],
      "contactPoint": {
        "@type": "ContactPoint",
        "telephone": "+81-3-1234-5678",
        "contactType": "customer service"
      }
    },
    {
      "@type": "ImageObject",
      "@id": "https://example.com/#logo",
      "url": "https://example.com/wp-content/uploads/logo.png",
      "width": 600,
      "height": 60
    },
    {
      "@type": "WebSite",
      "@id": "https://example.com/#website",
      "url": "https://example.com",
      "name": "Acme Corporation Blog",
      "inLanguage": "ja",
      "publisher": {
        "@id": "https://example.com/#organization"
      },
      "potentialAction": {
        "@type": "SearchAction",
        "target": {
          "@type": "EntryPoint",
          "urlTemplate": "https://example.com/?s={search_term_string}"
        },
        "query-input": "required name=search_term_string"
      }
    },
    {
      "@type": "WebPage",
      "@id": "https://example.com/about/#webpage",
      "url": "https://example.com/about/",
      "name": "About Us - Acme Corporation",
      "isPartOf": {
        "@id": "https://example.com/#website"
      },
      "breadcrumb": {
        "@id": "https://example.com/about/#breadcrumb"
      }
    },
    {
      "@type": "BreadcrumbList",
      "@id": "https://example.com/about/#breadcrumb",
      "itemListElement": [
        {
          "@type": "ListItem",
          "position": 1,
          "name": "Home",
          "item": "https://example.com/"
        },
        {
          "@type": "ListItem",
          "position": 2,
          "name": "About Us"
        }
      ]
    }
  ]
}
```

## 4. FAQPage

When FAQ content is detected or the page type is set to FAQPage, the plugin extracts questions and answers into FAQPage schema.

FAQ コンテンツが検出された場合、またはページタイプが FAQPage に設定された場合、質問と回答を FAQPage スキーマに抽出します。

```json
{
  "@context": "https://schema.org",
  "@graph": [
    {
      "@type": "WebSite",
      "@id": "https://example.com/#website",
      "url": "https://example.com",
      "name": "Acme Support",
      "inLanguage": "ja"
    },
    {
      "@type": "FAQPage",
      "@id": "https://example.com/faq/#faqpage",
      "mainEntity": [
        {
          "@type": "Question",
          "name": "What are your business hours?",
          "acceptedAnswer": {
            "@type": "Answer",
            "text": "We are open Monday to Friday, 9:00 AM to 6:00 PM. We are closed on weekends and public holidays."
          }
        },
        {
          "@type": "Question",
          "name": "Do you offer international shipping?",
          "acceptedAnswer": {
            "@type": "Answer",
            "text": "Yes, we ship to over 50 countries worldwide. Shipping costs and delivery times vary by destination."
          }
        },
        {
          "@type": "Question",
          "name": "What is your return policy?",
          "acceptedAnswer": {
            "@type": "Answer",
            "text": "You can return any unused item within 30 days of purchase for a full refund. Please contact our support team to initiate a return."
          }
        }
      ],
      "isPartOf": {
        "@id": "https://example.com/#website"
      }
    },
    {
      "@type": "WebPage",
      "@id": "https://example.com/faq/#webpage",
      "url": "https://example.com/faq/",
      "name": "Frequently Asked Questions",
      "isPartOf": {
        "@id": "https://example.com/#website"
      }
    }
  ]
}
```

For WooCommerce products, the plugin generates Product schema with offers, brand, ratings, and gallery images.

WooCommerce 商品では、オファー、ブランド、評価、ギャラリー画像を含む Product スキーマを生成します。

## Validation / 検証

You can validate your JSON-LD output using:

JSON-LD 出力を検証するには以下のツールを使用してください：

- [Google Rich Results Test](https://search.google.com/test/rich-results)
- [Schema.org Validator](https://validator.schema.org/)

The plugin includes a built-in AEO/GEO diagnostic tool in **Settings → AEO Schema** that checks for missing required properties.

プラグインには **設定 → AEO Schema** に組み込みの AEO/GEO 診断ツールがあり、必須プロパティの欠落をチェックします。
