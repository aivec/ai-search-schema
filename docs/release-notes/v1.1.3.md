# v1.1.3 リリースノート

**リリース日**: 2026-02-03

## 概要

WordPress.org 審査対応として、Pro ライセンス/試用要素の削除、外部サービスの明記、短い説明文の追加、パッケージング見直しを行いました。スラッグ運用を `aivec-ai-search-schema` に統一しています。

## 変更内容

### 審査対応・コンプライアンス

- Pro ライセンス管理 UI/ロジックを WP.org 版から削除
- 外部サービス（Google Maps Geocoding API / OpenStreetMap Nominatim）を readme に明記
- WordPress.org で不要な `load_plugin_textdomain()` 呼び出しを削除
- WordPress.org 用アセット（スクリーンショット/アイコン）をパッケージから除外

### 技術的変更

- ヘルパー関数プレフィックスを `avc_ais_` に統一
- スラッグに合わせてインストールパスやバンドル名を更新

### ドキュメント

- readme の短い説明文を追加
- External Services セクションを追加

## 変更ファイル（抜粋）

- `aivec-ai-search-schema.php`（バージョン更新）
- `includes/functions.php`（ヘルパー関数のプレフィックス変更）
- `templates/admin-settings.php` / `assets/js/admin-settings.js`（ライセンスUI削除）
- `readme.txt` / `docs/readme.txt.tpl`（Short description + External Services）
- `gulpfile.js` / `bundle`（パッケージ名更新）

## 互換性

- 既存機能に変更なし（Pro/ライセンス関連は WP.org 版から削除）
- 言語ファイルを再生成
