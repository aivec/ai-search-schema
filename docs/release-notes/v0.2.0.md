# v0.2.0 - llms.txt Generation

**Release Date:** 2025-12-22

## New Features

### llms.txt Generation
- Added llms.txt generator to help AI systems understand your site structure
- Auto-generates content from pages and posts with titles and excerpts
- Editable via Settings â†’ AI Search Schema
- Served at `/llms.txt` with proper content-type headers
- Can be enabled/disabled from settings
- Rewrite rules are flushed on plugin activation/deactivation

## Technical Details

- New class: `AI_Search_Schema_Llms_Txt`
- New settings card in admin UI
- AJAX handler for regenerating content from site data
- Filter: `ai_search_schema_llms_txt_default_content` for customizing auto-generated content
- Options: `ai_search_schema_llms_txt` (content), `ai_search_schema_llms_txt_enabled` (toggle)

## Compatibility

- WordPress 6.0+
- PHP 8.0+
- Tested up to WordPress 6.7
