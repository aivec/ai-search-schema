# v0.9.0 - WP.org Release Candidate

Release date: 2024-12-24

## Overview

This release prepares AI Search Schema for WordPress.org submission. It includes comprehensive documentation updates, support policy clarification, and security reporting guidelines.

## New Features

### Support Infrastructure
- **GitHub Issue Templates**: Added structured bug report and feature request templates
- **Security Policy**: Added SECURITY.md with vulnerability reporting guidelines
- **Support Policy in readme.txt**: Clear documentation of supported and unsupported use cases

### Documentation Updates
- **readme.txt Improvements**:
  - Added llms.txt FAQ entry
  - Added SEO plugin compatibility FAQ
  - Comprehensive Support Policy section with bug report guidelines
  - Added `llms-txt` tag
