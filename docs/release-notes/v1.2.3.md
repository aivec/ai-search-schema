# v1.2.3 リリースノート

**リリース日**: 2026-03-04

## 概要

WP.org Plugin Check 対応として、テンプレート内の HTML 属性出力を適切にエスケープしました。

## 変更内容

### WP.org コンプライアンス

- テンプレート内の class/style 属性出力を `esc_attr()` でエスケープ
- disabled 属性を WordPress の `disabled()` ヘルパーに置き換え

## 変更ファイル

- `templates/wizard/wizard-page.php` - class 属性エスケープ
- `templates/wizard/steps/step-1-basics.php` - style 属性エスケープ
- `templates/wizard/steps/step-4-hours.php` - disabled() ヘルパー使用
- `templates/wizard/steps/welcome.php` - class 属性エスケープ
- `templates/admin-settings.php` - class 属性エスケープ（4箇所）
- `aivec-ai-search-schema.php` - バージョン 1.2.3
- `package.json` - バージョン 1.2.3
- `readme.txt` - Stable tag 1.2.3

## 互換性

- WordPress 6.0+
- PHP 8.0+
