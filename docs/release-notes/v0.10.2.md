# v0.10.2 Release Notes

## Dev ビルド改善

### 新機能
- **Test Manager リンク追加** (dev モード専用)
  - 設定ページ右下にフローティングボタンを表示
  - `tools/test-manager.php` への直接リンク
  - DEV バッジ付きで視認性を確保
  - `is_dev_mode()` メソッドで dev ビルドを検出

### テスト仕様更新
- `test-spec.json` を v0.10.0 に更新
- メニュー名を「AEO Schema」から「AI Search Schema」に変更
- llms.txt 機能テストを追加 (UI-120〜123, FN-090〜093)
- ライセンス機能テストを追加 (UI-130〜134, FN-100〜102)
- テスト総数: 170件 (16件追加)

### バグ修正
- `readme.txt` の生成出力との同期を修正
- PHPCS 警告を修正 (長い行、未使用パラメータ)

### 技術的変更
- `composer.lock` を `composer.json` と同期
- `class-ai-search-schema-settings.php`: `is_dev_mode()` メソッド追加
- `templates/admin-settings.php`: Test Manager リンク UI 追加

### 互換性
- WordPress 6.0以上
- PHP 8.0以上
