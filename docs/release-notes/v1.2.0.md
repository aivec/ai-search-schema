# v1.2.0 リリースノート

**リリース日**: 2026-03-03

## 概要

WordPress.org 審査の最終対応として、vendor/ ディレクトリと独自アップデート機能を完全に削除しました。プラグインは PSR-4 フォールバックオートローダーで動作し、WP.org 準拠の単体完結型パッケージになりました。

## 変更内容

### WP.org コンプライアンス（重要）

- `vendor/` ディレクトリを配布ZIPから完全除外（PSR-4 フォールバックオートローダー使用）
- `src/License/` ディレクトリを削除（Pro版専用の独自アップデートクライアント）
- スクリプトハンドルをプレフィックス化: `ais-settings` → `avc-ais-settings`
- スクリプトハンドルをプレフィックス化: `ais-wizard` → `avc-ais-wizard`
- ウィザードページスラッグを変更: `ais-wizard` → `avc-ais-wizard`

### ビルドシステム

- `.distignore` を更新: `/vendor`, `/src/License` を除外対象に追加
- `gulpfile.js` を更新: `composer.json`, `package.json`, `package-lock.json` をコピー対象から除外
- GitHub Actions ワークフローを更新: rsync 除外設定を WP.org 準拠に変更

### 削除された機能

以下は Pro 版（別配布）でのみ提供され、無料版には含まれません：
- Pro版専用の独自アップデート機能
- Pro版専用のライセンス検証クライアント
