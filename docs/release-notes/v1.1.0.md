# v1.1.0 - WordPress.org Review Compliance

Release date: 2025-01-21

## Overview

This release addresses all feedback from the WordPress.org plugin review process, ensuring compliance with directory guidelines.

## Breaking Changes

### Prefix Changes
All internal prefixes have been changed from `ai_search_schema_` to `avc_ais_` to comply with WordPress.org naming requirements (minimum 4-character unique prefix).

**Affected items:**
- Constants: `AI_SEARCH_SCHEMA_*` → `AVC_AIS_*`
- Options: `ai_search_schema_*` → `avc_ais_*`
- Transients: `ai_search_schema_*` → `avc_ais_*`
- AJAX actions: `ai_search_schema_*` → `avc_ais_*`
- Nonces: `ai_search_schema_*` → `avc_ais_*`
- Post meta: `_ai_search_schema_meta` → `_avc_ais_meta`
- Filter/Action hooks: `ai_search_schema_*` → `avc_ais_*`

### Plugin Name Change
Plugin renamed from "AI Search Schema" to "Aivec AI Search Schema" for distinctiveness in the WordPress.org directory.

## Security Improvements

- Added `sanitize_text_field()` to nonce verification in metabox save handler
- Added ABSPATH protection to main plugin file and admin settings template
- Moved inline styles to `wp_add_inline_style()` for proper enqueuing

## Documentation

- Added comprehensive External Services section to readme.txt documenting:
  - Google Maps Geocoding API usage and privacy policy
  - OpenStreetMap Nominatim API usage and terms of service
  - License validation API (Pro version - future)

## Build Changes

- WordPress.org directory assets (icons, screenshots, banners) are now excluded from the plugin zip file
- These assets should be uploaded separately via SVN after plugin approval

## Upgrade Notes

**Important:** This version includes breaking changes to internal naming conventions. If you have:
- Custom code referencing plugin constants, options, or hooks
- Database entries with old option names

You may need to migrate your data or update your custom code. The old option keys will not be automatically migrated.

### For Developers

If you were using any of the following in custom code, update your references:

```php
// Constants (old → new)
AI_SEARCH_SCHEMA_VERSION → AVC_AIS_VERSION
AI_SEARCH_SCHEMA_DIR     → AVC_AIS_DIR
AI_SEARCH_SCHEMA_URL     → AVC_AIS_URL
AI_SEARCH_SCHEMA_FILE    → AVC_AIS_FILE

// Options (old → new)
'ai_search_schema_options' → 'avc_ais_options'

// Filters (old → new)
'ai_search_schema_google_maps_api_key' → 'avc_ais_google_maps_api_key'
'ai_search_schema_wizard_steps'        → 'avc_ais_wizard_steps'
```
