# v0.10.0 Release Notes

## Phase 4: Pro Version Foundation

### New Features
- **License Management Class** (`includes/class-ai-search-schema-license.php`)
  - License key storage, validation, and status management
  - `is_pro()` method to check Pro license status
  - `validate()` method for license validation (currently stub)
  - AJAX handlers for license activation/deactivation

- **Pro Features Manager** (`includes/class-ai-search-schema-pro-features.php`)
  - Pro feature registration and management
  - Pro feature list retrieval
  - Upgrade notification display

- **License Card in Settings**
  - License status display
  - License key input field
  - Activation/deactivation buttons
  - Pro feature preview (multi-location, custom templates, etc.)

### Technical Changes
- `ai-search-schema.php`: Added Pro features manager initialization
- `ai-search-schema.php`: Added license option null filter
- `admin-settings.js`: Added AJAX handlers for license activation/deactivation
- `admin.scss`: Added license UI styles
- Added Japanese translations

### Compatibility
- WordPress 6.0+
- PHP 8.0+
