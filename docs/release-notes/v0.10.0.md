# v0.10.0 Release Notes

## Phase 4: Pro版基盤準備

### 新機能
- **ライセンス管理クラス追加** (`includes/class-ai-search-schema-license.php`)
  - ライセンスキーの保存・検証・ステータス管理
  - `is_pro()` メソッドでPro版の有効/無効を判定
  - `validate()` メソッドでライセンス検証（現在はスタブ）
  - AJAX経由でのライセンス有効化/無効化

- **Pro機能管理クラス追加** (`includes/class-ai-search-schema-pro-features.php`)
  - Pro機能の登録・管理
  - Pro機能リストの取得
  - アップグレード通知の表示

- **設定画面にライセンスカード追加**
  - ライセンスステータス表示
  - ライセンスキー入力フィールド
  - 有効化/無効化ボタン
  - Pro機能プレビュー（マルチロケーション、カスタムテンプレート等）

### 技術的変更
- `ai-search-schema.php`: Pro機能マネージャーの初期化を追加
- `ai-search-schema.php`: ライセンスオプションのnullフィルターを追加
- `admin-settings.js`: ライセンス有効化/無効化のAJAXハンドラーを追加
- `admin.scss`: ライセンスUI用のスタイルを追加
- 日本語翻訳を追加

### 互換性
- WordPress 6.0以上
- PHP 8.0以上

### 次のステップ
- ライセンス検証APIの実装
- Pro機能の実装（マルチロケーション等）
- WP.org申請準備（v1.0.0）
